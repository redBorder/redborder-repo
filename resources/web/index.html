redborder repo ng web site

This is the rpm packages repository for redborder ng.

If you want to add this repo to your Centos 7 distro, please run the next command:

yum install epel-release && rpm -ivh https://repo.redborder.com/devel/redborder-repo-0.0.6-1.el7.noarch.rpm


If you want to add this repo to your Rocky Linux 9 distro, please run the next command:

yum install epel-release && rpm -ivh https://repo.redborder.com/ng/rhel/9/x86_64/redborder-repo-0.0.6-1.el9.noarch.rpm
